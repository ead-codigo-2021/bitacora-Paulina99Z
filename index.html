¿EN QUE CONSISTE?

El curso trata el diseño de códigos de interfaz para distintos medios y soportes digitales, es decir, la capa Front-end dentro de un proyecto digital. Los alumnos aprenderán a desarrollar proyectos digitales enfocándose en cómo estos se componen y construyen a través de un código limpio, claro y ordenado, utilizando lenguajes tales como HTML, CSS, Javascript entre otros.

1Objetivos del curso

1.- Conocer y utilizar las herramientas técnicas para la implementación de proyectos digitales.
2.- Comprender las posibilidades técnicas y particularidades del medio digital.
3.- Comprender las relaciones entre contenidos, datos e interfaz.

2Entrega bitacora

1.- Trabajo individual para documentar y ejercitar aprendizajes.
2.- Se debe realizar de manera progresiva a lo largo del semestre.
3.- Cada semana agregaremos nuevos requerimientos, que se van acumulando.
4.- Se debe trabajar y publicar en repositorio de tarea en Github.
5.- Evaluación corresponderá a 35% de nota final.
6.- Co-evaluación de bitácoras corresponderá a 10% de nota final.

TIPS

HTML | Escribe todas las etiquetas en minúsculas


Es más fácil de leer y de escribir.
Una página se comprime mejor cuando se usan etiquetas en minúsculas. Bueno, vale, porque es que algunos servidores web las transfieren comprimidas.

CSS | Comprende qué es el modelo de caja y qué propiedades afectan sus dimensiones.


El modelo de caja hace referencia a los elementos del HTML que están contenidos en una caja rectangular, esta tiene una serie de propiedades que afectan como se muestran.
height, width, margin, padding y border

JS | Diferencia las variables dependiendo de su importancia


Se recomiendo colocar aquellas variables que son usadas durante todo el script en la cabecera del código, de esta manera siempre sabrás donde encontrar estas variables que son las que determinan el resultado de nuestro código.
