CLASE 9

JS y preprocesadores

|15 de Octubre 2021

Un preprocesador css es un programa que convierte un tipo de información en hojas de estilo css

Scss es posible anidar los selectores de los elementos descendentes, lo que permite visualizar mejor la jerarquía de los elementos.
Debe utilizarse cuidadosamente, ya que la especificidad de los elementos puede aumentar si darnos cuenta

Scss las variables de cada preprocesador cuenta con la alternativa de crear variables reutilizables, y deben estar declaradas (escritas) antes de ser utilizadas.

Scss operaciones una cualidad que consideran casi todos los preprocesadores es poder realizar operaciones matemáticas dentro
del valor de una propiedad. SCSS considera al menos +, -, *, /, y %

Scss funciones ofrece una larga lista de funciones que podemos utilizar para gestionar colores, hacer cálculos o programar funciones más complejas

Scss import es una de las principales ventajas de un preprocesador es la capacidad de importar otros archivos scss
(o css), lo que permite aislar estilos en hojas separadas.

Scss mixins es una forma de crear funciones propias, para agrupar estilos comunes que puedan generarse
a partir de variables. Podemos definir los parámetros y comportamientos internos a partir de ellos.
