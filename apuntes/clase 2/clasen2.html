html significa HyperText Markup Language


Se trabaja con etiquetas y estas pueden contener una dentro de otra, también existe la indentacion que es aplicar sucesivos
niveles de márgenes de acuerdo a la anidación de etiquetas que permite identificar con mayor
facilidad problemas de anidación o desbalance en etiquetas de cierre.


Se le otorgan atributos.


Se recomienda para nombres de archivos y carpetas escribir en minúsculas y utilizar solo letras, números y guiones.


La estructura base es head seguido de body


La semántica es aplicar la etiqueta apropiada según el sentido del contenido.


Las etiquetas semánticas neutras div o span
